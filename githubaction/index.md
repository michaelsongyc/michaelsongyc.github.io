# Github Pages Action (github 自动部署指令)


## github 自动部署流程
- 开发人员把代码push 到 github
- 触发 github 钩子（push）执行 github action预定好的工作流
- 安装依赖，打包构建
- 通过ssh 上传静态资源到服务器
- 可选性通知开发者
