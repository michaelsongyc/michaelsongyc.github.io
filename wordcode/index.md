# Word中如何插入代码块？


# 方式1
使用 [codeinword](http://www.codeinword.com/) 在线工具，选择code对应的语言后，生成copy，粘贴到word中即可。


# 方式2

使用 Visual Studio code.  VSCode 支持复制后直接粘贴到word即是着色后的代码高亮的效果。

注意：
- 用vscode 写完代码后，一定要保存为指定的格式文件(如.java)；
- 复制vscode 代码到word，完美。

因为微软自家的软件所用的字符格式是相通的，比如说你写工作邮件，想要从word 或者excel 复制内容到chrome 网页版的邮箱里，这时候你可能发现格式丢失，如果使用微软自家浏览器edge 或者 ie 就不会有这个问题。
