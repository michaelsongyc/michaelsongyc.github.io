# 作图工具-学习法

## Tool
### mermaid
https://mermaid.live
#### 费曼学习法
```mermaid
graph TD
    A[选择学习概念] --> B[向新手解释概念]
    B --> C{解释过程中是否卡住?}
    C -->|是| D[回头重新学习相关部分]
    C -->|否| E[简化解释，用自己的话和例子]
    D --> B
    E --> F[完成学习]
```
#### 主动回忆法
```mermaid
graph TD
    A[学习材料] --> B[不看材料回忆内容]
    B --> C[检查回忆结果]
    C --> D{是否完全记住?}
    D -->|否| E[针对未记住部分再次学习]
    D -->|是| F[完成学习]
    E --> B
```

#### 间隔重复法
```mermaid
graph TD
    A[学习新知识] --> B[1 天后复习]
    B --> C[3 天后复习]
    C --> D[7 天后复习]
    D --> E[21 天后复习]
    E --> F[定期复习强化记忆]
    F --> G[长期记忆形成]
```
